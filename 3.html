Javascript

A programming language based on Ecma International's prototype , it is a scripting languag. An interpreter is built into all web browsers, unless there is a special purpose. Today, it is one of the elements that make up the web along with HTML and CSS. If HTML is responsible for the basic structure of a web page and CSS is responsible for the design, JavaScript is responsible for making the web page work on the client side.
